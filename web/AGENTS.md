# 当前 Next.js 版本可能与你熟悉的版本不同

此版本包含破坏性变更，API、约定和目录结构可能与训练数据不同。编写代码前必须阅读 `node_modules/next/dist/docs/` 中对应的当前版本指南，并遵守弃用提示。

## Typography

- Visible UI text must not render below `12px` on desktop or mobile.
- Use `13px` or larger for body copy and form content when space permits; reserve `12px` for metadata, compact labels, and dense operational controls.
- Do not rely on the browser's default `<small>` scaling. Set or inherit an explicit readable size.
