# InterviewCopilot Web

本目录是使用 Next.js 官方 `create-next-app` 创建的 Web 应用。

## 启动开发服务

```powershell
npm run dev
```

浏览器访问 [http://localhost:3000](http://localhost:3000)。修改 `src/app/page.tsx` 后页面会自动更新。

## 常用命令

```powershell
npm run lint
npm run build
npm run start
```

使用 Next.js API 前先阅读当前安装版本自带的文档：`node_modules/next/dist/docs/`。不要根据旧版本经验猜测接口。

官方资料：

- [Next.js 文档](https://nextjs.org/docs)
- [Next.js 学习教程](https://nextjs.org/learn)
